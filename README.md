# Munchkin Companion
This website will be a simple companion app for the Munchkin card game. It will provide players with a simple display of their current level and gear, and will provide an easy way to CRUD house rules. There will also be a TV mode in jackbox-like style, where all users stats will be displayed, as well as the current combat numbers.

This is in no way affiliated with the Munchkin game or the creator Steve Jackson Games, and is just for fun.
